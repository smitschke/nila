Sicherheitsscanner wählen

Möchte man NoSQL-Injections durchführen, muss man zunächst herausfinden, ob die entsprechende Seite angreifbar ist. Dies kann man mit sog. Sicherheitsscannern überprüfen.

Dazu gibt es eine Reihe von Programmen, die im Internet zum Download angeboten werden:

  • Burp Suite
  • OWASP ZAP
  • Invicti (formerly Netsparker)
  • Acunetix
  • Indusface WAS
  • Intruder
  • Tenable Nessus
  • ImmuniWeb
  • Veracode
  • Metaspoilt
  • G2 Deals
  • OpenVAS
  • Detectify
  • Amazon Inspector
  • IBM Security QRadar
  • uvm.

Die meisten der oben aufgeführten Anwendungen sind kostenpflichtig, mit Ausnahme von OWASP Zap und BurpSuite, das zumindest eine im Funktionsumfang reduzierte, kostenfreie Version zum Download bietet.

Da zur Verwendung von Burp Suite bereits viele Tutorials und eine gute Dokumentation existieren, wollen wir in dieser Lernumgebung ebenfalls diesem Tool widmen, wobei auch die anderen Anwendungen alle ihre Vor- und Nachteile bzw. ihren speziellen Einsatzzweck haben. Für unseren Einsatzzweck reicht jedoch die freie Version von Burp Suite vollkommen aus.

Folgende Liste soll aber noch einmal verdeutlichen, welche Aspekte generell bei der Auswahl von Sicherheitsscannern zu berücksichtigen sind:

  • Einfache Einrichtung und Konfiguration
  • Resistenz gegen Fehlalarme durch unüberprüfte Schwachstellen
  • Möglichkeiten des Loggings, der Berichterstellung und der Visualisierung
  • Kosten und Kundensupport

Die Software "Burp Suite"
Ein großer Vorteil von Burp Suite ist, dass mit dieser Software Sicherheitslücken jeglicher Art aufgedeckt werden können. Der Nachteil der sich wiederum daraus ergibt, ist dass dies nicht automatisch geschehen kann, sondern dass alle Tests manuell durchgeführt werden müssen.
Ein weiterer Nachteil ist, dass Burp Suite für weitergehende Abfragen manuell konfiguriert werden muss, damit der Datenverkehr zwischen Webserver und dem Browser abgefangen werden kann. Es handelt sich daher um eine Plattform, die eher für Personen mit technischem Fachwissen geeignet ist.

Generell stellt Burp Suite folgende Kernfunktionen bereit:

  • Web application crawling
  • Zeitgesteuerte, automatische Scans (in der kostenpflichtigen Version)
  • Manipulation von Requests
  • Burp Intruder zur Automatisierung benutzerdefinierter Angriffe

Hinweis: Den Downloadlink für Burp Suite findest du unter https://portswigger.net/burp/communitydownload
