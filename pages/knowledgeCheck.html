Wissenscheck

  1. In welcher Form speichern relationale Datenbanken (SQL-Datenbanken) ihre Daten?
    Antwort anzeigen
    In tabellarischer Form.
  2. Was ist ein großer Vorteil rationaler Datenbanken?
    Antwort anzeigen
    Sie können die ACID-Eigenschaften einhalten.
  3. Wieso nutzen große Unternehmen NoSQL-Datenbanken statt SQL, obwohl sie dadurch den großen Vorteil aus der letzten Frage verlieren?
    Antwort anzeigen
    Wegen den Skalierungsgrenzen von SQL-Datenbanken.
  4. In welcher Form speichern relationale Datenbanken (SQL-Datenbanken) ihre Daten?
    Antwort anzeigen
    In tabellarischer Form.
  5. Zwischen welchen vier Arten von NoSQL-Datenbanken wird unterschieden?
    Antwort anzeigen
    Dokumentorientiert, Schlüssel/Wert-Paar, Spaltenorientiert und Graphorientiert.
  6. Was macht ein Angreifer, um einen Injektionsangriff auszuführen? Wieso ist das gefährlich?
    Antwort anzeigen
    Er fügt bösartigen Code über Suchfelder, Formaularfelder oder API ein. Der Angreifer hat Zugriff auf alle Daten in der Datenbank und diese verändern, löschen und neue Daten hinzufügen.
  7. Was sollte mit Benutzereingaben getan werden, bevor die Anfrage ausgeführt wird, um Angriffe zu verhindern?
    Antwort anzeigen
    Nicht gewünschte Symbole (wie $) entfernen, damit Operatoren nicht ausgeführt werden können. Eingabe in Datentyp String umwandeln.
  8. Was kann genutzt werden, um Sicherheitsmaßnahmen wie die der letzten Frage schnell und einfach in eine Datenbank einzufügen?
    Antwort anzeigen
    Libraries oder Packages der genutzten Datenbank.
